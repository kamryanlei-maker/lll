# Pronunciation AI Practice

This is a no-build Vercel website. Upload the CONTENTS of this folder to GitHub, not the folder itself.

Root files should be:
- index.html
- api/analyze-pronunciation.js
- README.md

Then deploy on Vercel. Add this Environment Variable:
- SPEECHACE_API_KEY = your Speechace API key

Optional Environment Variables:
- SPEECHACE_ENDPOINT = https://api.speechace.co or https://api2.speechace.com
- SPEECHACE_DIALECT = en-us or en-gb
